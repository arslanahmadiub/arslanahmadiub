Greenfield multi-role hospital management platform covering OPD/IPD, real-time doctor queue (SSE), pharmacy, and billing. Built with React, TypeScript, Node.js/Express, PostgreSQL/Prisma, deployed on AWS with an S3/CloudFront upload pipeline.
🔗 [smart-hms.draftly.uk](https://smart-hms.draftly.uk)

**📦 Smart Inventory — Multi-Tenant Retail ERP/POS**
Live retail SaaS for inventory, billing, and POS with FBR fiscal integration and Urdu voice POS via the Web Speech API. PERN stack with machine-fingerprinted licensing and multi-role RBAC.
🔗 [smart-erp.draftly.uk](https://smart-erp.draftly.uk)

**🎓 Smart LMS — Multi-Tenant Learning Management System**
Feature-flag-driven LMS supporting six user roles with SSE-based real-time notifications, built on a Supabase-hosted PostgreSQL backend.

🔗 [members.bkcarsauctions.com](https://members.bkcarsauctions.com)
