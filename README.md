I specialize in the **PERN/MERN stack** (TypeScript, React, Next.js, Node.js, PostgreSQL, MongoDB) with deep expertise in **AWS cloud infrastructure** (Lambda, EC2, S3, CloudFront, API Gateway, VPC), **real-time systems** (WebSocket, Socket.io, Redis, async queues), and modern tooling like tRPC and Drizzle ORM. Verified **Toptal Top 3%** engineer with a track record of building and fully automating end-to-end SaaS platforms — from multi-tenant ERPs and healthcare billing systems to real-time auction platforms — for clients across the **US, UAE, and UK**.
